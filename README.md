# 🚀 Amazing Kanban

A modern and responsive Kanban Board built using HTML, CSS, and JavaScript. It helps users organize tasks efficiently through an intuitive drag-and-drop interface.

## 🌐 Live Demo

https://amazing-kanban-board.netlify.app/

## ✨ Features

* Create new tasks
* Drag and drop tasks between columns
* Organize tasks into To Do, In Progress, and Done
* Responsive and user-friendly design
* Modern UI with smooth interactions
* Task management workflow

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/amazing-kanban.git
```

2. Open the project folder

```bash
cd amazing-kanban
```

3. Open `index.html` in your browser.
